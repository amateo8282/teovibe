  <action>
    1. `cd teovibe && bundle add faraday`로 Faraday gem 추가

    2. `app/services/` 디렉토리 생성 (없으면)

    3. `app/services/payment_service.rb` 생성 (RESEARCH.md Pattern 3 기반):
       - TOSS_CONFIRM_URL = "https://api.tosspayments.com/v1/payments/confirm"
       - `initialize(secret_key)`: secret_key를 인스턴스 변수로 저장
       - `confirm(payment_key:, order_id:, amount:)`:
         - Faraday.new로 커넥션 생성 (request :json, response :json)
         - Authorization 헤더: `"Basic #{Base64.strict_encode64("#{@secret_key}:")}"` (콜론 포함 주의)
         - POST body: `{ paymentKey: payment_key, orderId: order_id, amount: amount }`
         - 성공: `{ success: true, data: response.body }` 반환
         - 실패: Faraday::Error rescue -> `{ success: false, error: e.message }` 반환
       - `require "base64"` 상단에 추가

    4. secret_key 접근: `Rails.application.credentials.dig(:toss_payments, :secret_key)` 사용
       - 주의: credentials에 toss_payments 키가 없으면 nil 반환됨. 사용자가 `rails credentials:edit`으로 설정해야 함.
       - 테스트 키: `test_sk_...` 형태 (RESEARCH.md 참조)
  </action>
  <verify>
    `cd teovibe && bundle list | grep faraday`로 faraday gem 설치 확인.
    `cd teovibe && ruby -c app/services/payment_service.rb`로 구문 오류 없음 확인.
    `cd teovibe && bin/rails runner "ps = PaymentService.new('test_key'); puts ps.respond_to?(:confirm)"` 출력이 true.
  </verify>
  <done>

  </done>
